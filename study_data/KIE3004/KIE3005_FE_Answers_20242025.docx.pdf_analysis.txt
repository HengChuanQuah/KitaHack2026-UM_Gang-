### **Question 1: Second-Order Linear Non-Homogeneous Differential Equation**

**PYQ Question:** Solve the second-order linear non-homogeneous differential equation:
y'' - 4y' + 4y = e^(2x)

**Solution Guide:**

**Concept from Lecture Notes:**
To solve a non-homogeneous linear differential equation of the form ay'' + by' + cy = g(x), we find the general solution y(x) = y_c(x) + y_p(x), where y_c(x) is the complementary solution (solution to the homogeneous equation) and y_p(x) is a particular solution.

**Step 1: Find the Complementary Solution (y_c(x))**
The homogeneous equation is: y'' - 4y' + 4y = 0.
The characteristic equation is: r² - 4r + 4 = 0.
Factoring gives: (r - 2)² = 0.
This yields a repeated real root: r₁ = r₂ = 2.

**From Lecture Notes (Case of Repeated Real Roots):** For repeated real roots r, the complementary solution is y_c(x) = C₁e^(rx) + C₂xe^(rx).
Therefore, y_c(x) = C₁e^(2x) + C₂xe^(2x).

**Step 2: Find a Particular Solution (y_p(x))**
The non-homogeneous term is g(x) = e^(2x).
**From Lecture Notes (Method of Undetermined Coefficients):**
If g(x) is of the form e^(ax), and 'a' is a root of the characteristic equation, the initial guess for y_p(x) needs to be multiplied by x until it is linearly independent of the terms in y_c(x).
Here, a = 2, which is a repeated root of the characteristic equation (multiplicity 2).
Initial guess: A e^(2x) -- This is part of y_c(x).
Second guess: Ax e^(2x) -- This is also part of y_c(x).
Third guess: Ax² e^(2x) -- This is linearly independent.
So, let y_p(x) = Ax² e^(2x).

Now, find the first and second derivatives of y_p(x):
y_p'(x) = A(2x e^(2x) + x² (2e^(2x))) = A(2xe^(2x) + 2x²e^(2x))
y_p''(x) = A( (2e^(2x) + 4xe^(2x)) + (4xe^(2x) + 4x²e^(2x)) )
y_p''(x) = A(2e^(2x) + 8xe^(2x) + 4x²e^(2x))

Substitute y_p, y_p', and y_p'' into the original non-homogeneous equation:
A(2e^(2x) + 8xe^(2x) + 4x²e^(2x)) - 4 * A(2xe^(2x) + 2x²e^(2x)) + 4 * A(x²e^(2x)) = e^(2x)

Expand and group terms:
e^(2x) [ 2A ] + xe^(2x) [ 8A - 8A ] + x²e^(2x) [ 4A - 8A + 4A ] = e^(2x)
e^(2x) [ 2A ] + xe^(2x) [ 0 ] + x²e^(2x) [ 0 ] = e^(2x)
2A e^(2x) = e^(2x)

Comparing coefficients: 2A = 1, so A = 1/2.
Therefore, y_p(x) = (1/2)x²e^(2x).

**Step 3: Form the General Solution**
The general solution is y(x) = y_c(x) + y_p(x).
y(x) = C₁e^(2x) + C₂xe^(2x) + (1/2)x²e^(2x).

### **Question 2: Vector Calculus (Gradient and Directional Derivative)**

**PYQ Question:** Given a scalar field φ(x, y, z) = x²yz + 3z³, find:
a) The gradient of φ (∇φ).
b) The directional derivative of φ in the direction of vector **v** = [1, 2, -2] at the point P(1, 1, 1).

**Solution Guide:**

**Concept from Lecture Notes:**
*   **Gradient:** For a scalar field φ(x, y, z), the gradient ∇φ is a vector field defined as ∇φ = (∂φ/∂x) **i** + (∂φ/∂y) **j** + (∂φ/∂z) **k**. It points in the direction of the maximum rate of increase of φ.
*   **Directional Derivative:** The directional derivative of φ in the direction of a unit vector **u** is given by D_u φ = ∇φ ⋅ **u**.

**Part a) Find the gradient of φ (∇φ)**

First, find the partial derivatives of φ with respect to x, y, and z:
∂φ/∂x = ∂/∂x (x²yz + 3z³) = 2xyz
∂φ/∂y = ∂/∂y (x²yz + 3z³) = x²z
∂φ/∂z = ∂/∂z (x²yz + 3z³) = x²y + 9z²

Now, assemble the gradient vector:
∇φ = [2xyz, x²z, x²y + 9z²]

**Part b) Find the directional derivative**

**Step 1: Calculate ∇φ at the point P(1, 1, 1)**
Substitute x=1, y=1, z=1 into ∇φ:
∇φ(1, 1, 1) = [2(1)(1)(1), (1)²(1), (1)²(1) + 9(1)²]
∇φ(1, 1, 1) = [2, 1, 1 + 9]
∇φ(1, 1, 1) = [2, 1, 10]

**Step 2: Find the unit vector **u** in the direction of **v** = [1, 2, -2]**
**From Lecture Notes:** A unit vector **u** is found by **u** = **v** / |**v**|.
First, find the magnitude of **v**:
|**v**| = √(1² + 2² + (-2)²) = √(1 + 4 + 4) = √9 = 3.

Now, find the unit vector **u**:
**u** = [1/3, 2/3, -2/3]

**Step 3: Calculate the directional derivative D_u φ = ∇φ ⋅ **u****
D_u φ = [2, 1, 10] ⋅ [1/3, 2/3, -2/3]
D_u φ = (2 * 1/3) + (1 * 2/3) + (10 * -2/3)
D_u φ = 2/3 + 2/3 - 20/3
D_u φ = (2 + 2 - 20) / 3
D_u φ = -16/3

**Answer:**
a) ∇φ = [2xyz, x²z, x²y + 9z²]
b) The directional derivative of φ at P(1, 1, 1) in the direction of **v** is -16/3.

### **Question 3: Laplace Transforms to Solve an Initial Value Problem**

**PYQ Question:** Use the Laplace transform to solve the initial value problem:
y' + 2y = cos(t), with y(0) = 0.

**Solution Guide:**

**Concept from Lecture Notes:**
The Laplace Transform is a powerful tool for solving linear ordinary differential equations, especially initial value problems. Key properties include:
*   L{y'(t)} = sY(s) - y(0)
*   L{y(t)} = Y(s)
*   L{cos(at)} = s / (s² + a²)
We solve by transforming the ODE into an algebraic equation in the 's' domain, solving for Y(s), and then finding the inverse Laplace transform.

**Step 1: Apply the Laplace Transform to both sides of the equation**
L{y' + 2y} = L{cos(t)}
L{y'} + 2L{y} = L{cos(t)}

**From Lecture Notes (Laplace Transform Formulas):**
Using L{y'(t)} = sY(s) - y(0) and L{cos(at)} = s / (s² + a²) (here a=1):
(sY(s) - y(0)) + 2Y(s) = s / (s² + 1²)

**Step 2: Substitute initial conditions**
Given y(0) = 0:
(sY(s) - 0) + 2Y(s) = s / (s² + 1)
sY(s) + 2Y(s) = s / (s² + 1)

**Step 3: Solve for Y(s)**
Factor out Y(s):
Y(s)(s + 2) = s / (s² + 1)
Y(s) = s / [(s + 2)(s² + 1)]

**Step 4: Perform Partial Fraction Decomposition for Y(s)**
**From Lecture Notes (Partial Fraction Decomposition):** For terms like (s + a)(s² + b²), use the form A/(s + a) + (Bs + C)/(s² + b²).
s / [(s + 2)(s² + 1)] = A/(s + 2) + (Bs + C)/(s² + 1)

Multiply by (s + 2)(s² + 1):
s = A(s² + 1) + (Bs + C)(s + 2)
s = As² + A + Bs² + 2Bs + Cs + 2C
s = (A + B)s² + (2B + C)s + (A + 2C)

Equate coefficients:
Coefficient of s²: A + B = 0  (Eq. 1)
Coefficient of s: 2B + C = 1  (Eq. 2)
Constant term: A + 2C = 0  (Eq. 3)

From (Eq. 1): B = -A
Substitute B into (Eq. 2): 2(-A) + C = 1  => -2A + C = 1 (Eq. 4)
From (Eq. 3): A = -2C
Substitute A into (Eq. 4): -2(-2C) + C = 1 => 4C + C = 1 => 5C = 1 => C = 1/5

Now find A and B:
A = -2C = -2(1/5) = -2/5
B = -A = -(-2/5) = 2/5

So, Y(s) = (-2/5)/(s + 2) + ((2/5)s + 1/5)/(s² + 1)
Y(s) = (-2/5) * 1/(s + 2) + (2/5) * s/(s² + 1) + (1/5) * 1/(s² + 1)

**Step 5: Find the Inverse Laplace Transform to get y(t)**
**From Lecture Notes (Inverse Laplace Transform Formulas):**
*   L⁻¹{1/(s - a)} = e^(at)
*   L⁻¹{s/(s² + a²)} = cos(at)
*   L⁻¹{a/(s² + a²)} = sin(at)

y(t) = L⁻¹{Y(s)}
y(t) = (-2/5)L⁻¹{1/(s + 2)} + (2/5)L⁻¹{s/(s² + 1)} + (1/5)L⁻¹{1/(s² + 1)}

y(t) = (-2/5)e^(-2t) + (2/5)cos(t) + (1/5)sin(t)

### **Question 4: Fourier Series**

**PYQ Question:** Find the Fourier Series for the function f(x) = x over the interval [-π, π].

**Solution Guide:**

**Concept from Lecture Notes:**
The Fourier Series for a function f(x) defined on the interval [-L, L] is given by:
f(x) = a₀/2 + Σ[n=1 to ∞] (a_n cos(nπx/L) + b_n sin(nπx/L))
where the coefficients are:
a₀ = (1/L) ∫[-L to L] f(x) dx
a_n = (1/L) ∫[-L to L] f(x) cos(nπx/L) dx
b_n = (1/L) ∫[-L to L] f(x) sin(nπx/L) dx

For the interval [-π, π], L = π.
So the formulas simplify to:
a₀ = (1/π) ∫[-π to π] f(x) dx
a_n = (1/π) ∫[-π to π] f(x) cos(nx) dx
b_n = (1/π) ∫[-π to π] f(x) sin(nx) dx

**From Lecture Notes (Even and Odd Functions):**
*   If f(x) is an even function (f(-x) = f(x)), then b_n = 0.
*   If f(x) is an odd function (f(-x) = -f(x)), then a₀ = 0 and a_n = 0.
*   For an odd function, ∫[-L to L] f(x) dx = 0.
*   For an even function, ∫[-L to L] f(x) dx = 2 ∫[0 to L] f(x) dx.

**Step 1: Determine if f(x) is even or odd**
f(x) = x
f(-x) = -x
Since f(-x) = -f(x), f(x) is an odd function.

**Step 2: Calculate coefficients a₀ and a_n**
Since f(x) = x is an odd function:
a₀ = (1/π) ∫[-π to π] x dx = 0
a_n = (1/π) ∫[-π to π] x cos(nx) dx = 0 (because x is odd, cos(nx) is even, so x cos(nx) is odd)

**Step 3: Calculate coefficient b_n**
b_n = (1/π) ∫[-π to π] x sin(nx) dx

Since x sin(nx) is an even function (odd * odd = even), we can write:
b_n = (2/π) ∫[0 to π] x sin(nx) dx

**From Lecture Notes (Integration by Parts):** ∫u dv = uv - ∫v du
Let u = x, dv = sin(nx) dx
Then du = dx, v = -cos(nx)/n

b_n = (2/π) [ [-x cos(nx)/n] from 0 to π - ∫[0 to π] (-cos(nx)/n) dx ]
b_n = (2/π) [ [-π cos(nπ)/n - (-0 cos(0)/n)] + (1/n) ∫[0 to π] cos(nx) dx ]
b_n = (2/π) [ -π cos(nπ)/n + (1/n) [sin(nx)/n] from 0 to π ]
b_n = (2/π) [ -π cos(nπ)/n + (1/n²) (sin(nπ) - sin(0)) ]

**From Lecture Notes (Trigonometric Values):**
cos(nπ) = (-1)^n
sin(nπ) = 0 for integer n
sin(0) = 0

Substitute these values:
b_n = (2/π) [ -π (-1)^n / n + (1/n²) (0 - 0) ]
b_n = (2/π) [ -π (-1)^n / n ]
b_n = -2 (-1)^n / n
b_n = 2 (-1)^(n+1) / n

**Step 4: Write the Fourier Series**
Since a₀ = 0 and a_n = 0, the Fourier series consists only of sine terms:
f(x) = Σ[n=1 to ∞] (b_n sin(nx))
f(x) = Σ[n=1 to ∞] (2 (-1)^(n+1) / n) sin(nx)

Expanding the first few terms:
For n=1: b₁ = 2(-1)²/1 = 2 => 2 sin(x)
For n=2: b₂ = 2(-1)³/2 = -1 => -sin(2x)
For n=3: b₃ = 2(-1)⁴/3 = 2/3 => (2/3) sin(3x)
For n=4: b₄ = 2(-1)⁵/4 = -1/2 => -(1/2) sin(4x)

So, f(x) = 2 sin(x) - sin(2x) + (2/3)sin(3x) - (1/2)sin(4x) + ...
