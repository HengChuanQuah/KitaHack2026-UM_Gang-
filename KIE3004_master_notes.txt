1 Introduction  
Electrical circuits containing resistors, inductors, and capacitors , also known as RLC 
circuits, play a crucial  role in many applications of engineerin g, particularly in signal 
processing, power systems, and control engineering. The unsteady -state behaviour of 
such circuits is controlled  by differential equations which is often unable to  be solved 
analytically for real life engineering problems. As a result, numerical methods become 
essential tools for analysing circuit responses over time.  
This assignment focuses on the numerical solution of the unsteady -state current flow in 
an RLC circuit driven by a 20 V  source is investigated, with parameters 𝐿 = 1 H, 𝐶 = 0.1 F, 
and 𝑅1 = 𝑅2 = 5 Ω. The given circuit model leads to a second -order differential equation, 
which is first transformed into a system of first -order differential equations. Numerical 
techniques are then applied to solve the system subject to specified boundary conditions, 
namely zero initial current at 𝑡=0s and an approximately steady-state current of 2 A at 
𝑡=10s. 
To achieve this, numerical integration methods are implemented using MATLAB, without 
relying on built -in solvers, in order to strengthen understanding of the underlying 
numerical algorithms. The shooting method is employed to estimate the unknown initial 
conditions by iteratively adjusting them until the required boundary condition at the final 
time is satisfied. Through this approach, the assignment demonstrates how numerical 
analysis techniques can be effectively applied to solve boundary value problems a rising 
from real -world electrical systems.

3 Assuming the current at initial time (t=0) is 0 A and 
current reaches steady state approximately 2 A in the 
circuit at final time of 10 s, develop the algorithm to

(at t=0 s) using MATLAB.  
 
 
clc;clear;  
 
%% Time settings  
dt = 0.01;  
t = 0:dt:10;  
N = length(t);  
 
%% Initial condition for current  
x1_0 = 0;                 % i(0) = 0 A  
 
%% Initial guess for di/dt  
x2_0 = 1;                 % initial guess  
tolerance = 0.01;         % acceptable error  
error = 1;  
 
%% Iterative shooting algorithm  
while error > tolerance  
 
    % Initialize state variables  
    x1 = zeros(1, N);  
    x2 = zeros(1, N);  
 
    x1(1) = x1_0;  
    x2(1) = x2_0;  
 
    % Forward Euler integration  
    for k = 1:N-1 
        x1(k+1) = x1(k) + dt*x2(k);  
        x2(k+1) = x2(k) + dt*(40 - 7*x2(k) - 20*x1(k));  
    end 
 
    % Compute error at final time  
    error = abs(x1(end) - 2); 
 
    % Adjust initial slope  
    if x1(end) < 2  
        x2_0 = x2_0 + 0.1;  
    else 
        x2_0 = x2_0 - 0.1; 
    end 
end 
 
%% Display result  
disp(['Estimated initial di/dt = ' , num2str(x2_0)]);  
 
%% Plot result  
plot(t, x1, 'LineWidth' , 2); 
xlabel('Time (s)' ); 
ylabel('Current (A)' ); 
title('Numerical Solution of RLC Circuit Current' ); 
grid on;

state current flow of the RLC circuit using a technique known as the "Shooting Method." 
This approach is necessitated by the nature of the problem, which is defined as a 
boundary value problem rather than a standard initial value problem. While the initial 
current at t=0 is explicitly given as 0 A, the initial rate of change of current (𝑑𝑖
𝑑𝑡) is unknown. 
Instead, the system is constrained by a final boundary condition where the current must 
reach a steady state of approximately 2 A at t=10s. Consequently, the algorithm must 
iteratively estimate the unknown initial slope until the simulation aligns with the required 
final state.  
The algorithm begins by defining the simulation parameters, establishing a time domain 
from 0 to 10 seconds with a step size ( dt) of 0.01 seconds. This fine discretization is 
essential to minimize cumulative errors during numerical integration. The state variables 
are initialized with 𝑥1 (current) set to 0 A, satisfying the given initial condition. The second 
state variable, 𝑥2(the derivative𝑑𝑖

Inside the loop, the algorithm utilizes the Forward Euler method to solve the system of

Upon the completion of each 10 -second simulation cycle, the algorithm evaluates the

4 Conclusion  
In conclusion, this report has successfully demonstrated the application of numerical 
analysis techniques to solve the unsteady -state current behaviour  of an RLC circuit. By 
reducing the governing second -order differential equation into a state -space 
representation, the system was effectively modelled  for computational processing 
without relying on in -built MATLAB solvers.  
The implementation of the Shooting Method, utilizing a custom Forward Euler integration 
algorithm, allowed for the accurate determination of the unknown initial condition , 
specifically the rate of change of current (𝑑𝑖
𝑑𝑡) at t = 0. The iterative algorithm successfully 
converged, identifying the precise initial slope required for the system to transition from 
an initial current of 0 A to the required steady -state value of 2 A.  
The resulting simulation confirms the dynamic response of the circuit parameters ( L=1 H, 
R=5Ω , C=0.1 F) to the 20V source, showing a clear transition from the transient state to 
stability within the 10 -second interval. Ultimately, this study reinforces the importance 
of numerical methods in electrical engineering, proving that fundamental algorithmic 
logic can be used  to solve complex boundary value problems where analytical solutions 
or standard initial value approaches are insufficient.
